# nyph-app

This repo implements the recording form used for the BSBI's Nyph recording form. See https://nyph.bsbi.org/

## Getting started

Clone this repository and install its dependencies:

```bash
git clone https://github.com/BSBI/nyph-app
cd nyph-app
npm install
```
